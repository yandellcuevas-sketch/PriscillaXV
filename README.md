# 💙 XV Años – Priscilla

Una experiencia web romántica e interactiva creada especialmente para los XV años de Priscilla.

## 🌐 Ver experiencia en vivo

**[https://yandellcuevas-sketch.github.io/PriscillaXV/](https://yandellcuevas-sketch.github.io/PriscillaXV/)**

## ✨ Características

- Portada animada con estrellas y partículas
- Marco elegante azul y plateado para el video especial
- Carta romántica con efecto máquina de escribir
- Animación de corazones, besos y destellos
- Diseño mobile-first compatible con Android e iPhone
- Vibración suave en dispositivos compatibles
- Música ambiental opcional

## 📁 Estructura de archivos

```
PriscillaXV/
├── index.html     ← Experiencia completa (HTML + CSS + JS)
├── video.mp4      ← Video especial de XV años
├── musica.mp3     ← Música ambiental (opcional)
└── README.md
```

## 📱 Compatibilidad

- ✅ Android (Chrome, Samsung Browser)
- ✅ iPhone (Safari)
- ✅ GitHub Pages (HTTPS)
